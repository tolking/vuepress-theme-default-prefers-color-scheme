# Migration from v1

TIP

If and only if you have configured defaultTheme in the v1 version, you need to know this

module.exports = {
  theme: 'default-prefers-color-scheme',
  themeConfig: {
-   defaultTheme: 'dark',
+   overrideTheme: 'dark',
  },
- postcss: {
-   plugins: [
-     require('css-prefers-color-scheme/postcss'),
-     require('autoprefixer')
-   ]
- }
}
